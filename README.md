# MPM-Py

A Material Point Method implementation using Python: MPM-Py.

## Installation

```git
git clone https://github.com/fabricix/MPM-Py.git
```

## Requirements

* Python 3.7.4 or superior

* Matplotlib 3.3.4 or superior
